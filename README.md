## Contexte

Déploiement chez **Sanitoral**, société internationale spécialisée dans les soins 
bucco-dentaires, d'un tableau de bord de suivi de projets pour Sophie, cheffe de 
projet au sein du service PMO. Le besoin portait sur trois axes : suivre l'avancement 
des projets, identifier les retards et contrôler les performances.

## Objectifs

- Formaliser les besoins via une note de cadrage
- Automatiser le nettoyage des données via Power Query
- Créer un dashboard interactif de suivi PMO
- Former les équipes à la mise à jour autonome hebdomadaire

## Ce que j'ai fait

- Note de cadrage et Product Strategy Canvas
- Nettoyage et transformation automatisée des données (Power Query)
- Développement des visualisations interactives (Power BI)
- Intégration d'un onglet documentation (procédure de mise à jour, modèle de données)
- Restitution client en storytelling data
